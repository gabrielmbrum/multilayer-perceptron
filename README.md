# MLP (Multilayer Perceptron)

By using basic perceptron cells (repo w/ the concepts and implementation [here](https://github.com/gabrielmbrum/perceptron-and-adaline)), it's possible to combine a bunch of perceptrons together to build a large MLP.

With MLPs, we can solve way more complex problems, such as non-linear problems, multiclass classification, and so on.

This repo contains an MLP Class with a very flexible architecture. You can use it for different input/output dimensions and set up various hidden layers (each with a different number of neurons).

Check out the `report.pdf` for an analysis of the results across three different datasets. Each dataset was tested with variations in regularization, activation functions, optimizers, topologies, learning rates, and epochs!
